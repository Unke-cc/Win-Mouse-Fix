# MVP 手动检查清单

每个发布候选版本都应完整执行本清单。记录 Windows 版本、程序版本、鼠标型号、测试人员和日期；任何失败项都要附上复现步骤。

## 准备

- 使用至少一只有中键、按键 4 和按键 5 的鼠标。
- 准备普通窗口、浏览器、文件管理器、全屏视频和一款全屏游戏。
- 准备 Windows 10 22H2 x64 和 Windows 11 x64 测试机；两台机器均不单独安装 AutoHotkey。
