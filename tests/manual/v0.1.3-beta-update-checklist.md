# Win Mouse Fix 0.1.3 Beta 更新检查清单

本清单覆盖 0.1.3 Beta 的滚动组合键和触摸板保护改动，并包含必要的回归检查。

## 滚动组合键

- [ ] 默认组合保持为：Shift 水平滚动、Alt 快速滚动、Win 精确滚动、Ctrl 放大或缩小。
- [ ] 在四个输入框分别录入 `Ctrl+Shift`、`Alt+Win` 等组合，界面显示规范顺序。
- [ ] 按住完整组合并滚动，确认对应功能生效；只按其中一部分时不触发组合功能。
- [ ] 输入 A、F1、Ctrl+A 等普通键或完整字符快捷键，确认不会写入配置并显示提示。
- [ ] 按 Delete 或 Backspace 清除组合，确认该项显示为“关闭”。
- [ ] 为两个功能设置相同组合，确认后设置的项目被关闭并显示提示。
- [ ] 导出、导入和切换配置后，组合键内容保持不变。
- [ ] 读取旧版只包含单个修饰键的配置，确认仍能正常使用。

## 触摸板与滚轮

- [ ] 使用触摸板连续上下滑动页面 5 秒，不出现热键频率提示，页面保持系统滚动。
- [ ] 使用实体鼠标快速滚轮，确认不会被错误暂停，普通滚轮仍可用。
- [ ] 使用实体鼠标按住中键或侧键并滚动，确认融合动作不受影响。
- [ ] 使用实体鼠标按住 Ctrl、Shift 等组合并滚轮，确认滚动功能正常。
