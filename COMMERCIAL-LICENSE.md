# Win Mouse Fix 商业许可说明

Win Mouse Fix `0.1.0` 的默认许可只允许个人非商业使用。以下情况需要在使用前取得胡文凯的单独书面许可：

- 企业、组织或个体经营者在工作中使用；
- 将软件或修改版本用于收费产品、收费服务或营利活动；
- 向客户、员工、合作方或公众分发程序；
- 使用 Win Mouse Fix 名称、图标或其他品牌元素开展商业活动。

商业许可可以根据实际情况约定使用人数、设备数量、分发范围、费用、支持、更新和期限。本文只说明申请范围，本身不授予商业使用权。

申请时请通过项目仓库 <https://github.com/Unke-cc/Win-Mouse-Fix> 或仓库所有者公开的联系方式，说明：

1. 个人或组织名称；
2. 计划用途和使用人数；
3. 是否需要向他人分发；
4. 预计使用期限和支持需求。

未来版本可以采用不同的个人收费或商业条款。购买或取得某一版本的许可，不自动获得其他版本的权利。
